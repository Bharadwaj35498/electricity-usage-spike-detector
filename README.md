# Electricity / Data Usage Spike Detector

This project detects abnormal spikes in electricity or mobile data usage using
time-series analysis and unsupervised machine learning.

## Techniques Used
- Rolling Mean & Standard Deviation
- Z-Score Anomaly Detection
- Isolation Forest

## Tools
Python, Pandas, NumPy, Scikit-learn, Matplotlib

## How to Run
```bash
python main.py
